# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-04-29

### 新增

- 🖥️ 基于 Wails + Vue 3 的图形化界面
- 📋 扫描任务管理功能
- 🎯 目标管理功能
- 🔍 指纹管理功能
- 🛡️ POC 管理功能
- 📊 结果分析功能
- ⚙️ 系统设置功能

### 修复

- 🔒 修复并发安全问题（添加互斥锁保护全局 map）
- 🔧 修复 goroutine 泄漏问题
- 🔌 修复连接泄漏问题
- 🔐 移除硬编码的 AES 密钥
- 🛡️ 修复路径遍历安全漏洞
- 📝 修复文件权限过宽问题
- ⚠️ 修复类型断言 panic 风险
- ⚠️ 修复反射调用 panic 风险

### 优化

- ⚡ GC 频率从 10 秒调整为 5 分钟
- ⚡ 正则表达式预编译优化
- 🎨 前端错误处理改进
- 📝 添加 .gitignore 文件

### 安全

- 🔐 AES 密钥改为从环境变量读取
- 🛡️ 添加 POC 删除路径验证
- 📝 文件权限从 0666 改为 0644
